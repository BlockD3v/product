# Trade.xyz built a stock exchange on Hyperliquid. So did the oil traders.

## Angle

HIP-3 went permissionless October 13, 2025. The 500k HYPE deployer stake (~$20M) was supposed to be a barrier — instead it's a filter that's attracting *serious* operators. Trade.xyz launched 24/7 perps on TSLA/AAPL/NVDA/AMZN + a synthetic Nasdaq index. Oil-focused builders pushed commodity OI to $1.43B. This is the canonical "Hyperliquid as exchange-as-a-service" piece nobody has written.

## Why now

- Aggregated HIP-3 OI: ~$340M (and growing)
- "Oil's favorite exchange just beat Robinhood" frame already in TheStreet
- Profile gap: no comprehensive list/profile of current HIP-3 deployers exists

## Key evidence

- 500k HYPE stake (~$20M) per deployer
- First 3 markets free; subsequent slots via Dutch auction
- Fee structure: ~0.07–0.09% taker, split 50/50 deployer/protocol
- Validators can stake-vote slash for malicious markets; 30-day cooldown post-halt
- Coverage: equities, commodities, FX, RWAs, prediction markets, crypto tail assets
