# Builder codes — Hyperliquid's affiliate program nobody talks about

## Angle

Builder codes let any app developer attach per-order fee splits, user-revocable, on every fill they route. Embedded in mobile wallets, terminals, and AI trading agents. There is almost no how-to content in market — this should be the canonical playbook. SEO + developer-funnel double-win.

## Why now

- "hyperliquid api" search volume +50% MoM (DataForSEO)
- AI agent trading is a hot narrative; builder codes are the monetization layer
- Hyperliquid's own docs are reference-grade but not narrative — gap for a builder-friendly walkthrough

## Key evidence

- Per-order fee specification (max flexibility)
- User must approve max builder fee per builder
- Revocable any time (trust-minimized)
- Composes with HIP-3 (builders can deploy *and* monetize order flow)
